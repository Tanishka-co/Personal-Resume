# Personal-Resume

A clean and responsive personal resume website built from scratch using **HTML** and **CSS**, designed to present my skills, projects, and experience in a professional and visually appealing way.

---

## 🔗 Live Demo

>https://tanishka-co.github.io/Personal-Resume/

---

## 📁 Project Structure

```
Personal-Resume/
├── Resume.html      # Main resume page
└── style.css        # Styling and layout
```

---

## ✨ Features

- Clean, minimal design
- Fully responsive layout
- Sections for skills, projects, and experience
- Pure HTML & CSS — no frameworks or dependencies
- Fast loading and easy to customize

---

## 🛠️ Built With

- **HTML5**
- **CSS3**

---

## 🚀 How to Run Locally

1. Clone the repository:
```bash
   git clone https://github.com/Tanishka-co/Personal-Resume.git
```
2. Open `Resume.html` in your browser — that's it!
